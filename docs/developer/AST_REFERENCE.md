# Condition AST Reference

This document provides a comprehensive reference for the Abstract Syntax Tree (AST) nodes generated by the `yaraConditionParser.mjs` and consumed by the `yaraConditionsMatch.mjs` evaluator.

## Basic Types

### Literals
Values directly embedded in the source.
```javascript
{ type: 'boolean', value: true }
{ type: 'number', value: 42 }
{ type: 'string', value: "some_string" }
```

### Identifiers
References to global variables or loop variables.
```javascript
{ type: 'identifier', name: "filesize" }
{ type: 'identifier', name: "entrypoint" }
{ type: 'identifier', name: "i" } // Loop variable
```

### Rule Reference
Referencing another rule's status.
```javascript
{ type: 'ruleIdentifier', name: "SuspiciousRule" }
```

## Operations

### Logical
```javascript
{ type: 'and', left: Node, right: Node }
{ type: 'or', left: Node, right: Node }
{ type: 'not', operand: Node }
{ type: 'defined', operand: Node }
```

### Arithmetic
```javascript
{ type: 'add', left: Node, right: Node }
{ type: 'subtract', left: Node, right: Node }
{ type: 'multiply', left: Node, right: Node }
{ type: 'divide', left: Node, right: Node }
{ type: 'modulo', left: Node, right: Node }
```

### Unsigned Bitwise (32-bit)
Note: The evaluator forces operands to unsigned 32-bit integers (`>>> 0`) before operation.
```javascript
{ type: 'bitwiseAnd', left: Node, right: Node }
{ type: 'bitwiseOr', left: Node, right: Node }
{ type: 'bitwiseXor', left: Node, right: Node }
{ type: 'bitwiseNot', operand: Node }
{ type: 'shiftLeft', left: Node, right: Node }
{ type: 'shiftRight', left: Node, right: Node }
```

### Comparison
```javascript
{ type: 'equal', left: Node, right: Node }
{ type: 'notEqual', left: Node, right: Node }
{ type: 'lessThan', left: Node, right: Node }
{ type: 'greaterThan', left: Node, right: Node }
{ type: 'lessThanOrEqual', left: Node, right: Node }
{ type: 'greaterThanOrEqual', left: Node, right: Node }
```

## Strings

### String Logic
```javascript
{ type: 'contains', left: Node, right: Node }
{ type: 'icontains', left: Node, right: Node } // Case-insensitive
{ type: 'startswith', left: Node, right: Node }
{ type: 'istartswith', left: Node, right: Node }
{ type: 'endswith', left: Node, right: Node }
{ type: 'iendswith', left: Node, right: Node }
{ type: 'matches', left: Node, right: Node } // Regex match (left matches right)
{ type: 'iequals', left: Node, right: Node } // Case-insensitive equality
```

### String References
```javascript
// $a
{ type: 'stringIdentifier', identifier: "$a" } 

// #a (Count)
{ type: 'stringCount', identifier: "$a" }

// @a[index] (Offset)
{ type: 'stringOffset', identifier: "$a", index: Node (default 0) }

// !a[index] (Length)
{ type: 'stringLength', identifier: "$a", index: Node (default 0) }
```

## Loops & Quantifiers

### For Loop
```javascript
{
  type: 'for',
  quantifier: 'all' | 'any' | 'none' | { type: 'number', value: N } | { type: 'percentage', value: N },
  variable: string | null, // 'i' or null for 'them'
  set: Node, // range, stringSet, or null
  condition: Node
}
```

### Quantified Expression
Equivalent to `for X of them`.
```javascript
{
  type: 'quantified',
  quantifier: 'all' | 'any' | 'none' | { type: 'number', value: N } | { type: 'percentage', value: N },
  items: Array<string> // List of string identifiers e.g., ['$a', '$b']
}
```

### Sets
```javascript
// (1..10)
{ type: 'range', start: Node, end: Node }

// ($a, $b, $c)
{ type: 'stringSet', items: Array<string> }
```

## Data & Modules

### Data Access
Reads integer at offset.
```javascript
{
  type: 'dataAccess',
  dataType: 'uint8' | 'int16' | ..., 
  offset: Node,
  endian: 'LE' | 'BE'
}
```

### Module Access
```javascript
// math.pi
{ 
  type: 'memberAccess', 
  object: Node, 
  property: string 
}

// pe.sections[0]
{ 
  type: 'arrayAccess', 
  object: Node, 
  index: Node 
}

// math.max(1, 2)
{ 
  type: 'moduleFunction', 
  module: string, 
  function: string, 
  args: Array<Node> 
}
```

## Position

```javascript
// $a at 0x100
{ type: 'at', identifier: string|Node, offset: Node }

// $a in (0..100)
{ type: 'inRange', identifier: string, range: Node }

// $a within 10 of $b
{ type: 'within', identifier: string, distance: Node, reference: string }
```
